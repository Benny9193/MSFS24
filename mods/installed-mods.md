## Installation Notes

### Community Folder Location

**Microsoft Store Version:**
```
%LOCALAPPDATA%\Packages\Microsoft.Limitless_8wekyb3d8bbwe\LocalCache\Packages\Community
```

**Steam Version:**
```
%APPDATA%\Microsoft Flight Simulator 2024\Packages\Community
```

### Mod Sources

| Source | URL | Notes |
|--------|-----|-------|
| Flightsim.to | https://flightsim.to | Largest free mod repository |
| MSFS Marketplace | In-sim | Official paid/free content |
| SimMarket | https://secure.simmarket.com | Paid add-ons |
| Orbx | https://orbx.to | Premium scenery |
| PMDG | https://pmdg.com | Premium aircraft |

### Installation Tips

1. **Backup first** - Copy Community folder before major changes
2. **One at a time** - Install mods individually to isolate issues
3. **Check compatibility** - Verify mod is for MSFS 2024 (not 2020)
4. **Read instructions** - Some mods require specific installation
5. **Keep organized** - Use subfolders for mod categories
6. **Update regularly** - Check for mod updates after sim updates

---

## Troubleshooting

| Issue | Possible Cause | Solution |
|-------|---------------|----------|
| CTD on startup | Incompatible mod | Remove recent additions |
| Missing textures | Incomplete install | Reinstall mod |
| Performance drop | Too many add-ons | Disable unused mods |
| Conflicting mods | Same files modified | Remove one mod |

### Mod Conflict Testing

1. Remove all mods from Community folder
2. Test if issue persists
3. Add mods back one at a time
4. Identify conflicting mod
