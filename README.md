# MSFS24 Documentation

Personal documentation, guides, and resources for Microsoft Flight Simulator 2024.

## Directory Structure

```
MSFS24/
├── guides/           Game guides and tutorials
│   ├── beginner/     Getting started tutorials
│   ├── advanced/     IFR, navigation, complex systems
│   └── aircraft/     Aircraft-specific guides
│
├── mods/             Add-ons and modifications
│   ├── liveries/     Aircraft paint schemes
│   ├── scenery/      Airport and scenery add-ons
│   ├── aircraft/     Third-party aircraft
│   └── utilities/    Helper tools and plugins
│
├── reference/        Quick reference materials
│   ├── checklists/   Aircraft checklists
│   ├── charts/       Navigation charts, airport diagrams
│   └── keybinds/     Control mappings and profiles
│
├── configs/          Configuration files
│   ├── graphics/     Graphics presets
│   ├── controls/     Controller profiles
│   └── community/    Community folder backups
│
├── assets/           Media files
│   ├── screenshots/  In-game screenshots
│   └── images/       Guide images, diagrams
│
└── notes/            Personal notes and flight logs
```

## Quick Links

- **MSFS Official**: https://www.flightsimulator.com/
- **Community Forums**: https://forums.flightsimulator.com/
- **Flightsim.to**: https://flightsim.to/ (mods and add-ons)

## Usage Tips

- Keep mod backups in `mods/` before installing to Community folder
- Store controller profiles in `configs/controls/` for easy restoration
- Use `notes/` for flight planning and personal flight logs
