# Cessna 172 Skyhawk Checklist

## Aircraft Info
- **Type:** Single Engine Piston
- **Model:** Cessna 172 Skyhawk (G1000)
- **Cruise Speed:** 124 KTAS
- **Vr (Rotate):** 55 KIAS
- **Vy (Best Climb):** 74 KIAS
- **Vref (Approach):** 65 KIAS
- **Vs0 (Stall, Full Flaps):** 40 KIAS
- **Vs1 (Stall, Clean):** 48 KIAS

---

## Pre-Flight

- [ ] Weather briefing obtained
- [ ] NOTAMs checked
- [ ] Weight & balance calculated
- [ ] Fuel quantity verified
- [ ] Flight plan filed (if required)

---

## Before Engine Start

- [ ] Parking brake - SET
- [ ] Seats, belts, harnesses - ADJUST & LOCK
- [ ] Fuel selector - BOTH
- [ ] Avionics master - OFF
- [ ] Circuit breakers - CHECK IN
- [ ] Master switch (BAT & ALT) - ON
- [ ] Fuel quantity indicators - CHECK
- [ ] Avionics cooling fan - AUDIBLE
- [ ] Flaps - UP
- [ ] Beacon light - ON

---

## Engine Start

- [ ] Mixture - RICH
- [ ] Throttle - OPEN ¼ inch
- [ ] Propeller area - CLEAR
- [ ] Ignition - START
- [ ] Oil pressure - CHECK (green within 30 sec)
- [ ] Throttle - 1000 RPM
- [ ] Avionics master - ON
- [ ] Flaps - CHECK operation
- [ ] Radios/GPS - SET

---

## Before Taxi

- [ ] Altimeter - SET (local QNH)
- [ ] Heading indicator - ALIGN with compass
- [ ] Flight instruments - CHECK
- [ ] Autopilot - CHECK (if installed)
- [ ] NAV/GPS flight plan - LOADED
- [ ] Taxi clearance - OBTAINED
- [ ] Brakes - TEST during initial taxi

---

## Run-Up (Before Takeoff)

- [ ] Parking brake - SET
- [ ] Doors & windows - CLOSED & LATCHED
- [ ] Flight controls - FREE & CORRECT
- [ ] Flight instruments - CHECK
- [ ] Fuel selector - BOTH
- [ ] Mixture - RICH (below 3000 ft)
- [ ] Throttle - 1800 RPM
- [ ] Magnetos - CHECK (max 125 RPM drop, max 50 RPM differential)
- [ ] Carburetor heat - CHECK (RPM drop, then OFF)
- [ ] Engine instruments - GREEN
- [ ] Ammeter - CHECK charging
- [ ] Suction gauge - CHECK (4.5-5.4)
- [ ] Throttle - 1000 RPM
- [ ] Throttle friction - ADJUST
- [ ] Strobe lights - ON
- [ ] Transponder - ALT
- [ ] Takeoff briefing - COMPLETE

---

## Takeoff (Normal)

- [ ] Flaps - 0° (or 10° for short field)
- [ ] Carburetor heat - OFF
- [ ] Landing light - ON
- [ ] Takeoff clearance - RECEIVED
- [ ] Brakes - RELEASE
- [ ] Throttle - FULL
- [ ] Engine instruments - CHECK
- [ ] Airspeed - ALIVE
- [ ] Rotate at 55 KIAS
- [ ] Climb at 74 KIAS (Vy)

---

## Climb

- [ ] Airspeed - 70-85 KIAS
- [ ] Throttle - FULL
- [ ] Mixture - RICH (lean above 3000 ft)
- [ ] Engine temps - MONITOR
- [ ] Flaps - UP (above 200 ft AGL)
- [ ] Landing light - OFF (above 1000 ft AGL)

---

## Cruise

- [ ] Altitude - VERIFY
- [ ] Throttle - SET (2200-2400 RPM typical)
- [ ] Mixture - LEAN for best economy
- [ ] Trim - ADJUST for level flight
- [ ] Engine instruments - MONITOR
- [ ] Fuel management - MONITOR quantity

---

## Descent

- [ ] ATIS/AWOS - OBTAIN
- [ ] Altimeter - SET (local QNH)
- [ ] Mixture - ENRICH as required
- [ ] Carburetor heat - AS REQUIRED (apply before reducing power)
- [ ] Fuel selector - BOTH
- [ ] Approach briefing - COMPLETE

---

## Before Landing

- [ ] Seats, belts - SECURE
- [ ] Fuel selector - BOTH
- [ ] Mixture - RICH
- [ ] Carburetor heat - ON
- [ ] Landing light - ON
- [ ] Autopilot - OFF

---

## Landing (Normal)

- [ ] Airspeed - 65-75 KIAS (final approach)
- [ ] Flaps - AS REQUIRED
  - 10° - Below 110 KIAS
  - 20° - Below 85 KIAS
  - FULL - Below 85 KIAS (final)
- [ ] Trim - ADJUST
- [ ] Touchdown - MAIN WHEELS FIRST
- [ ] Brakes - AS REQUIRED

---

## After Landing

- [ ] Carburetor heat - OFF
- [ ] Flaps - UP
- [ ] Transponder - STBY
- [ ] Strobes - OFF
- [ ] Landing light - OFF (when clear of runway)

---

## Shutdown

- [ ] Parking brake - SET
- [ ] Throttle - IDLE (1000 RPM)
- [ ] Electrical equipment - OFF
- [ ] Avionics master - OFF
- [ ] Mixture - CUTOFF (engine stops)
- [ ] Ignition - OFF
- [ ] Master switch - OFF
- [ ] Fuel selector - LEFT or RIGHT
- [ ] Control lock - INSTALL

## Emergency Quick Reference

**Engine Failure on Takeoff:**
1. Pitch for 65 KIAS
2. Mixture, fuel, ignition - CHECK
3. Land straight ahead

**Engine Failure in Flight:**
1. Airspeed - 65 KIAS (best glide)
2. Select landing site
3. Troubleshoot: Fuel, mixture, carb heat, ignition
4. If no restart: Mayday call, transponder 7700
5. Secure engine before touchdown
