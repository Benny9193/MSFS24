## Scenic Flight Routes

### Hudson River VFR Corridor
```
KTEB → Hudson River → Statue of Liberty → KTEB
Distance: ~40 nm round trip
Aircraft: Any GA (helicopter ideal)
Altitude: At or below 1,100 ft
Highlights: Manhattan skyline, Statue of Liberty, GW Bridge
Note: RIGHT side of river northbound, LEFT southbound
Mandatory: 123.050 for position reports
```

### Cape Cod & Islands
```
KBED → KPVC (Provincetown) → KACK → KMVY → KHYA → KBED
Distance: ~180 nm
Aircraft: C172, Bonanza
Highlights: Cape Cod beaches, islands, lighthouses
```

### Maine Coast
```
KPWM → KRKD (Rockland) → KBHB (Bar Harbor) → KBGR
Distance: ~200 nm
Aircraft: Any GA
Highlights: Acadia National Park, rocky coastline
```

### Finger Lakes
```
KROC → KITH (Ithaca) → KELM (Elmira) → KBGM → KSYR
Distance: ~200 nm
Aircraft: Any GA
Highlights: Finger Lakes, gorges, wine country
```

### Fall Foliage (Vermont/NH)
```
KBDL → KLEB (Lebanon) → KMPV (Montpelier) → KBTV
Distance: ~180 nm
Aircraft: Any GA
Highlights: Green Mountains, covered bridges
Best time: Late September - October
```

## NYC Airspace Guide

### Class B Structure
```
KJFK, KLGA, KEWR share NYC Class B
Surface: Inner rings around each airport
Steps: Various floors from 1,500 - 7,000 ft
Ceiling: 7,000 ft
```

### VFR Corridors

**Hudson River Exclusion:**
- At or below 1,100 ft MSL
- 123.050 MHz (not ATC - traffic advisory)
- Stay right of river centerline
- Position reports required

**East River:**
- At or below 1,100 ft MSL
- More restricted than Hudson

### TFRs to Watch
- UN General Assembly (September)
- Presidential movements (frequent)
- Major events (parades, games)

## Boston Airspace

### Class B Structure
- Surface to 7,000 ft
- Inner ring: Surface - 7,000
- Outer shelves step down

### VFR Tips
- Stay below 1,500 ft along coast (under shelves)
- Request Class B transition if able
- KBED often easier entry point
