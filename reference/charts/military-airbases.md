## Famous Locations

### Edwards AFB (KEDW)
**"Where the Air Force Tests Tomorrow's Aircraft Today"**

| Runway | Length | Notes |
|--------|--------|-------|
| 04L/22R | 15,024 ft | Main |
| 04R/22L | 12,000 ft | Secondary |
| 06/24 | 7,000 ft | Auxiliary |
| Lakebed | 7+ miles | Multiple dry lakebed runways |

**Why It's Special:**
- Largest military base by area (301,000 acres)
- Dry Rogers Lake bed = unlimited runway options
- Chuck Yeager broke sound barrier here
- Space Shuttle emergency landing site
- X-15, SR-71, B-2, F-22, F-35 all tested here

**In MSFS:** Look for the lakebed runways - huge open expanse

## Scenic Military Routes

### Nevada Range Run
```
KLAS → KLSV (Nellis) → KXTA (Area 51) → KEDW
Distance: ~250 nm
Aircraft: F-22, F-16, F-35
Highlights: Nellis ranges, Groom Lake, Edwards
Note: In real life, don't fly over Area 51!
```

### California Test Corridor
```
KEDW → KNTD (Point Mugu) → KNZY (North Island) → KNKX (Miramar)
Distance: ~300 nm
Aircraft: Any military
Highlights: Mojave Desert, Channel Islands, San Diego harbor
```

### East Coast Fighter Sweep
```
KLFI (Langley) → KGSB (Seymour Johnson) → KNKT (Cherry Point) → KCHS
Distance: ~350 nm
Aircraft: F-22, F-15, F-35
Highlights: Virginia coast, Outer Banks
```

### Blue Angels Tour
```
KNPA (Pensacola) → KNJK (Key West) → KTPA → KNPA
Distance: ~500 nm round trip
Aircraft: F-18
Highlights: Gulf Coast, Keys, demonstration profiles
```

## Carrier Operations (Fictional in MSFS)

While MSFS doesn't have aircraft carriers, you can practice at:

### Carrier Landing Practice Fields

| ICAO | Name | Notes |
|------|------|-------|
| KNUW | NAS Whidbey OLF | Short runway, carrier sim |
| NALF | Fentress | Virginia, touch & go |
| KNJK | NAS Key West | Short approaches over water |

### Carrier Approach Profile
1. **Break** at 800 ft, 350 KIAS
2. **Downwind** - Configure, gear down
3. **Abeam** - Start descent
4. **Groove** - 600 ft, 3° glideslope
5. **Ball** - Call the ball
6. **Touchdown** - Hook zone
