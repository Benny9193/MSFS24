## Scenic Flight Routes

### Pacific Coast Highway (PCH)
```
KSMO → KSBA → KSBP → KMRY → KHAF → KSFO
Distance: ~350 nm
Aircraft: C172, Bonanza, SR22
Highlights: Malibu, Big Sur, Monterey Bay
Altitude: 2,000-3,500 ft for coastal views
```

### LA Basin Tour
```
KVNY → KSMO → KLAX → KLGB → KSNA → KONT → KVNY
Distance: ~120 nm
Aircraft: Any GA
Highlights: Hollywood sign, beaches, downtown LA
Note: Complex airspace - know your Bravos/Charlies
```

### Sierra Nevada
```
KFAT → KMMH → KTVL → KTRK → KSMF
Distance: ~250 nm
Aircraft: High-performance (terrain)
Highlights: Yosemite, Lake Tahoe, mountains
Caution: High density altitude, terrain
```

### Wine Country
```
KSQL → KAPC → KSTS → KUKI → KCCR
Distance: ~130 nm
Aircraft: Any GA
Highlights: Napa Valley, Sonoma, rolling hills
```

### Death Valley Run
```
KLAS → KBTY → L09 (Stovepipe Wells) → KBIH
Distance: ~200 nm
Aircraft: High-performance
Highlights: Lowest point in North America
Caution: Extreme heat, high DA in summer
```

## Airspace Notes

### Los Angeles Basin
- KLAX Class B: Surface to 12,500 ft
- Multiple Class C (BUR, SNA, ONT, LGB)
- VFR corridors available (Mini Route, Special Flight Rules)
- LA Special Flight Rules Area over LAX

### San Francisco Bay
- KSFO/KOAK Class B: Complex shared airspace
- KSJC Class C underneath
- Bay Tour route available (stay below Class B)

### San Diego
- KSAN Class B: Surface to 10,000 ft
- KNKX restricted areas nearby
- Mexican border - know your flight restrictions
