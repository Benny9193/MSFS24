## Scenic Flight Routes

### Miami to Key West (Island Hopping)
```
KMIA → KMTH (Marathon) → KEYW
Distance: ~130 nm
Aircraft: C172, TBM, Caravan
Highlights: Florida Keys, Seven Mile Bridge, coral reefs
```

### Gulf Coast Tour
```
KRSW → KAPF → KSRQ → KSPG → KTPA
Distance: ~140 nm
Aircraft: Any GA
Highlights: Beaches, downtown St. Pete, Tampa Bay
```

### Space Coast
```
KORL → KSFB → KTIX (Titusville) → KMLB
Distance: ~90 nm
Aircraft: Any GA
Highlights: Kennedy Space Center, Cape Canaveral
```

### Panhandle Beaches
```
KPNS → KVPS → KECP
Distance: ~90 nm
Aircraft: Any GA
Highlights: White sand beaches, Emerald Coast
```

## Notes

- All frequencies are simulated; verify in-sim for accuracy
- Military bases may have restricted airspace
- Key West (KEYW) has short runway (4800 ft) - plan accordingly
- KSPG (Albert Whitted) has challenging downtown approach over water
