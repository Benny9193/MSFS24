## Scenic Flight Routes

### Florida to Bahamas
```
KFLL → MYBS (Bimini) → MYNN (Nassau)
Distance: ~180 nm total
Aircraft: C172, Bonanza, Caravan
Highlights: Turquoise waters, island chains
Note: International flight - file flight plan
```

### Bahamas Island Hopping
```
MYNN → MYEH (Eleuthera) → MYES (Staniel Cay) → MYEG (Exuma)
Distance: ~150 nm
Aircraft: Any GA
Highlights: Swimming pigs at Staniel Cay, Thunderball Grotto
```

### Puerto Rico Circle
```
TJSJ → TJVQ (Vieques) → TJCP (Culebra) → TJBQ → TJSJ
Distance: ~200 nm
Aircraft: Any GA
Highlights: Bioluminescent bays, beaches
```

### Eastern Caribbean Chain
```
TAPA (Antigua) → TLPC (St. Lucia) → TBPB (Barbados)
Distance: ~350 nm
Aircraft: Multi-engine recommended
Highlights: Volcanic islands, rainforests
```

### St. Maarten Challenge
```
TNCM (Princess Juliana) - Famous beach landing approach
Highlights: Aircraft fly directly over Maho Beach
Challenge: Short runway, terrain, turbulence
```

## Over-Water Flying Tips

### Required Equipment (Real World)
- Life vests
- Life raft (beyond gliding distance)
- ELT (406 MHz recommended)
- VHF radio

### MSFS Considerations
- File flight plans (practice realism)
- Check weather at destination
- Know alternate airports
- Fuel planning critical - no gas stations mid-ocean

### Distances from Florida

| From KMIA to | Distance | Time (C172) |
|--------------|----------|-------------|
| MYBS (Bimini) | 50 nm | 0:30 |
| MYNN (Nassau) | 185 nm | 1:45 |
| MUHA (Havana) | 230 nm | 2:15 |
| TJSJ (San Juan) | 1,000 nm | 8:00+ |
| TNCM (St. Maarten) | 1,200 nm | 10:00+ |

## Weather Considerations

| Season | Conditions |
|--------|------------|
| Dec-Apr | Dry season, best flying |
| Jun-Nov | Hurricane season |
| Year-round | Trade winds (E/NE 10-20 kt typical) |
| Afternoons | Tropical buildups common |

### Hurricane Season
- Peak: August-October
- Check forecasts carefully
- Have alternate plans
- In sim: Use weather presets or live weather
