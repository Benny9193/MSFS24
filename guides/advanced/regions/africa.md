## Safari Flying

### East African Safari Circuit

**The Classic Route:**
```
HKJK (Nairobi) → HKML (Masai Mara) →
HTSE (Serengeti) → HTZA (Zanzibar) → HKJK
Distance: ~600 nm
Highlights: Big Five, Great Migration
Best time: July-October (migration)
```

### Safari Airstrips
| ID | Name | Location | Notes |
|----|------|----------|-------|
| HKML | Masai Mara | Kenya | Bush strip, wildlife |
| HTSE | Seronera | Serengeti, Tanzania | Dirt strip |
| HTBS | Selous | Tanzania | Remote |
| FBMN | Maun | Botswana | Okavango gateway |
| FVWN | Hwange | Zimbabwe | National park |

### Safari Flying Tips
- Wildlife on runways common
- Low pass to clear before landing
- Dirt/grass strips
- Fly early morning/late afternoon
- Heat = turbulence midday

## Iconic Destinations

### Egypt - Pyramids & Nile
```
HECA (Cairo) → Pyramids of Giza →
Luxor → Abu Simbel → HECA
Distance: ~1,200 nm round trip
Highlights: Ancient wonders
Note: Restricted airspace around monuments
```

### Kilimanjaro
```
HKJK (Nairobi) → HTKJ (Kilimanjaro) →
Circle Kilimanjaro → HKMO (Mombasa)
Distance: ~400 nm
Highlights: Africa's highest peak (19,341 ft)
Aircraft: High-performance for altitude
```

### Victoria Falls
```
FLLS (Livingstone) → Victoria Falls →
FVFA (Victoria Falls) → Zambezi River
Distance: ~50 nm
Highlights: Largest waterfall (by volume)
Best: Rainy season for full flow
```

### Okavango Delta
```
FBMN (Maun) → Okavango Delta →
Moremi → Chobe → FBMN
Distance: ~200 nm
Highlights: Largest inland delta
Best: Flood season (Jun-Aug)
```

### Cape Town
```
FACT (Cape Town) → Table Mountain →
Cape of Good Hope → Cape Agulhas → FACT
Distance: ~200 nm
Highlights: Where two oceans meet
Note: Strong winds common
```

## Bush Strip Operations

### Typical African Bush Strip
- Dirt or grass surface
- 800-1,500m length
- No facilities
- Wildlife hazards
- Limited/no fuel

### Bush Aircraft
| Aircraft | Role | Why |
|----------|------|-----|
| **Cessna 206** | Safari | Reliable, bush wheels |
| **Cessna Caravan** | Charter | Turbine, capacity |
| **Kodiak** | STOL | Modern bush plane |
| **King Air** | Executive | Twin turboprop |
| **Pilatus PC-6** | STOL | Legend |
| **Piper Cub** | Basic | Classic bush |

### Bush Flying Techniques
- STOL operations essential
- Soft field technique
- High approach (obstacles)
- Go-around always planned
- Fuel management critical

## Weather Patterns

### Climate Zones

| Zone | Area | Weather |
|------|------|---------|
| Saharan | North Africa | Extremely hot, dry |
| Sahel | Sub-Saharan | Hot, seasonal rain |
| Tropical | Central Africa | Hot, wet, thunderstorms |
| Savanna | East/Southern | Wet/dry seasons |
| Mediterranean | Far north/south | Mild, seasonal |
| Highland | Mountains | Cooler, variable |

### Seasons

**Dry Season (Best for Safari):**
- East Africa: June - October
- Southern Africa: May - October
- Vegetation low, animals at water

**Wet Season:**
- "Green season"
- Fewer tourists
- Challenging flying
- Thunderstorms

### Hazards
| Hazard | Where | When |
|--------|-------|------|
| Thunderstorms | Tropical/savanna | Wet season afternoon |
| Dust/harmattan | Sahel, West Africa | Dec-Feb |
| Heat | Everywhere | Midday |
| Sand storms | Sahara | Variable |
| ITCZ | Equatorial | Moves seasonally |

### ITCZ (Intertropical Convergence Zone)
- Band of thunderstorms
- Follows the sun north/south
- Can be severe
- Plan around it

## Unique Challenges

### Heat & Performance
- Surface temps 40-50°C (104-122°F)
- Density altitude extreme
- Fly early morning
- Calculate performance carefully

### Infrastructure
- Varies dramatically
- South Africa excellent
- Remote areas primitive
- Fuel availability uncertain

### Security
- Check current advisories
- Some areas restricted
- Civil unrest possible
- File flight plans

### Wildlife
- Animals on runways
- Elephants, zebra, antelope
- Birds (especially vultures)
- Low pass before landing

### Communications
- HF radio for remote areas
- Gaps in radar coverage
- Position reports critical

## Safari Flying Tips

### Best Practices
- Fly morning or evening (cooler, animals active)
- Always check strip conditions
- Carry satellite phone/beacon
- Know fuel availability
- Respect wildlife (altitude, distance)

### Photographic Flying
- Doors off (if possible)
- Morning light best
- High wing aircraft ideal
- Slow flight for shots
- Bank for unobstructed views
