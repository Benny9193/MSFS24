## Engine Start

### Ground Start (Cold & Dark)

1. **Battery** - ON
2. **APU** - ON
3. **Wait for APU ready**
4. **Left engine:**
   - Throttle - OFF → IDLE
   - Engine crank - L
   - Monitor RPM, EGT
   - Stable at idle
5. **Right engine:**
   - Throttle - OFF → IDLE
   - Engine crank - R
   - Monitor RPM, EGT
   - Stable at idle
6. **APU** - OFF (when stable)
7. **Generators** - ON
8. **Avionics** - Power up

### Carrier Start
- External power from carrier
- Same sequence, faster timeline
- Ready for launch within minutes

## Takeoff

### Field Takeoff

1. **Line up** - Centerline
2. **Flaps** - AUTO or HALF
3. **Trim** - Set
4. **Brakes** - HOLD
5. **Throttle** - MIL
6. **Engines** - Check matched
7. **Brakes** - RELEASE
8. **AB** - As required
9. **Rotate** - 130-150 KIAS
10. **Gear** - UP
11. **Flaps** - AUTO then UP

### Catapult Launch

1. **Taxi** to catapult
2. **Launch bar** - DOWN
3. **Spread wings** - Verify
4. **Flaps** - HALF
5. **Hook** - UP
6. **Holdback** engaged
7. **Run up** - MIL power
8. **Wipeout** - Full controls cycle
9. **Salute** - Signal ready
10. **Throttle** - AB (full)
11. **Head back** - Brace
12. **Launch!**
13. **Rotate** - Off deck
14. **Gear** - UP immediately
15. **Climb** - Positive rate

**Cat shot:** 0 to 150+ knots in 2-3 seconds!

## Carrier Operations

### Case I Recovery (Day VFR)

**The Pattern:**
1. **Overhead** - 800 ft, 350 KIAS
2. **Break** - At bow, 4G level turn
3. **Downwind** - 600 ft
   - Gear DOWN
   - Hook DOWN
   - Flaps FULL
4. **Abeam** - Start descent
5. **180° turn** - To final
6. **Groove** - 18 units AOA
7. **Ball call** - "[Callsign], Hornet, Ball, [fuel state]"
8. **Fly the ball** - Centered, on speed
9. **Touchdown** - In wires
10. **Full power** - Until trapped

### The Ball (IFLOLS)
```
        ○ ○ ○  ← High (fly down)
  GREEN DATUM LINE
        ● ○ ○  ← On glideslope (good)
  GREEN DATUM LINE
        ○ ○ ○  ← Low (fly up, WAVE OFF if red)
```

### AOA Indexer (Carrier Approach)
| Light | Meaning | Action |
|-------|---------|--------|
| Chevron (top) | SLOW | Add power |
| Donut (middle) | ON SPEED | Maintain |
| Chevron (bottom) | FAST | Reduce power |

**Target: 8.1 units AOA (donut)**

### Bolter
- Miss all wires
- Full power already applied
- Climb away
- Re-enter pattern

### Wave Off
- Mandatory if called by LSO
- Full power
- Wings level
- Climb straight ahead
- Do NOT land!

## Air Combat Maneuvering

### Rate vs Radius Fight

**F/A-18 is a RATE fighter:**
- Best sustained turn rate
- Wins two-circle fights
- Maintain 300-350 KIAS

**Two-Circle (Rate):**
- Both aircraft turn same direction
- Nose-to-nose passes
- Hornet advantage

**One-Circle (Radius):**
- Turn into each other
- Smallest radius wins
- Hornet can compete with high-AOA

### Slow-Speed Fight

The Hornet's party trick:
1. Bleed opponent's energy
2. High AOA (30°+)
3. Use lift vector
4. Point nose, snapshot

**Caution:** Vulnerable if you miss!

### Basic Maneuvers

**Defensive Break:**
1. Max G into threat
2. Dispense chaff/flares
3. 7-8G, pull nose down if needed
4. Unload, extend, reassess

**Lag Pursuit Roll:**
1. Opponent crossing
2. Pull up, roll toward
3. Lag pursuit to control overtake
4. Roll out in control zone

## Emergency Procedures

### Single Engine Failure
1. **Maintain control**
2. **Identify** - Dead engine
3. **Throttle** - Good engine MIL
4. **Secure** - Failed engine if needed
5. **Single-engine approach** - Fly slightly fast
6. **Land** - Longer runway preferred

### Dual Engine Failure
1. **Airspeed** - 190 KIAS (best glide)
2. **Altitude** - Trade for distance
3. **Restart attempt:**
   - Throttles - OFF
   - Windmill start above 250 KIAS
4. **No restart** - EJECT above 2,000 ft

### Hydraulic Failure
1. **Check HYD 1 & 2**
2. **Gain-override switch** - As required
3. **Land immediately**
4. **No-flap landing** if needed

### Ejection

**Martin-Baker SJU-17:**
- Zero-zero capability
- Pull either handle
- Canopy blows, then seat fires

**Decision:**
- Uncontrolled below 2,000 ft AGL = EJECT
- Controlled - attempt recovery
