## Engine Start

### APU Start

1. **Battery** - ON
2. **JFS (Jet Fuel Starter)** - START
3. **Wait** for JFS RPM
4. **Left throttle** - IDLE
5. **Monitor:**
   - RPM rising
   - FTIT (turbine temp)
   - Oil pressure
6. **Left engine stable** - Verify
7. **Right throttle** - IDLE
8. **Right engine stable** - Verify
9. **JFS** - OFF
10. **Generators** - ON
11. **Avionics** - Power sequence

### Air Start (Windmill)
- Above 250 KIAS
- Throttle - OFF then IDLE
- Ram air rotates engine
- Good for flameout restart

## Takeoff

### Standard Takeoff

1. **Line up** - Centerline
2. **Brakes** - HOLD
3. **Throttle** - MIL (both)
4. **Engines** - Check matched RPM
5. **Brakes** - RELEASE
6. **Afterburner** - Select as needed
7. **Maintain centerline** - Rudder
8. **Rotate** - 150-160 KIAS
9. **Lift off** - Smooth rotation
10. **Gear** - UP (positive climb)
11. **Accelerate** - 350 KIAS

### Max Performance Takeoff
- Full Zone 5 afterburner
- Rotation at 150 KIAS
- 60°+ pitch possible
- Vertical climb to 30,000 ft without leveling

## Air Combat Maneuvering

### Energy Fighting Philosophy

The F-15 is the **ultimate energy fighter:**
- Never give up energy advantage
- Vertical maneuvers preferred
- "First look, first shot, first kill"
- Extend and re-engage

### Vertical Tactics

**Zoom Climb:**
1. High speed entry (450+ KIAS)
2. Pull to vertical
3. Hold afterburner
4. Apex 40,000+ ft above entry

**Vertical Reversal:**
1. Opponent pursuing
2. Pull vertical, AB
3. At apex, roll and pull
4. Descend on opponent

**High Yo-Yo:**
1. Pull up above plane of fight
2. Roll inverted at apex
3. Pull down behind bandit
4. Arrive with energy

### Turn Performance

| Type | G | Speed |
|------|---|-------|
| Instantaneous | 9G | 450 KIAS |
| Sustained | 6G | 400 KIAS |
| Corner | Max rate | 450 KIAS |

### Basic Fighter Maneuvers

**Break Turn:**
1. Threat at 6 o'clock
2. Max G into threat
3. Dispense countermeasures
4. Maintain visual

**Guns Defense:**
1. Hard turn toward attacker
2. Force overshoot
3. Roll under/over
4. Reverse role

**Vertical Extension:**
1. Disengage by going vertical
2. Build altitude = energy
3. Re-engage with advantage

1. **Initial** - 1,500 AGL, 300+ KIAS

3. **Downwind** - Gear down below 280

## Emergency Procedures

### Single Engine Failure

1. **Maintain control** - Wings level
2. **Verify** - Which engine failed
3. **Good engine** - MIL power
4. **Failed engine:**
   - Throttle - OFF if fire/damage
   - Monitor temps
5. **Single-engine performance** - Still excellent
6. **Land** - Nearest suitable runway

**Note:** F-15 has excellent single-engine performance - can even take off on one engine!

### Dual Engine Failure

1. **Airspeed** - 220 KIAS (best glide)
2. **Altitude** - Maximize
3. **Restart attempt:**
   - Throttles - OFF
   - Airspeed - 250+ KIAS
   - Throttle - IDLE (one at a time)
4. **No restart** - EJECT above 2,000 ft AGL

### Fire

1. **Confirm** - Fire warning light
2. **Throttle** - Affected engine OFF
3. **Fire extinguisher** - As required
4. **Monitor** - Fire out?
5. **Land immediately** or EJECT

### Flight Control Malfunction

1. **Assess** - What's stuck?
2. **Reduce speed** - Lower loads
3. **CAS (Control Augmentation)** - Reset/override
4. **Land** - Straight-in if possible

### Ejection

**ACES II Seat:**
- Zero-zero capable
- Pull either handle
- Canopy blows → Seat fires → Chute deploys

**Decision:**
- Uncontrolled aircraft
- Fire uncontrollable
- Below 2,000 ft AGL with problems = EJECT

## Aerobatics

### Loop
1. Entry: 450 KIAS
2. 4-5G pull
3. Maintain back pressure
4. Over top ~150 KIAS
5. Pull through
6. Exit same altitude

### Immelmann
1. Entry: 450 KIAS
2. Pull to vertical
3. At apex, roll 180°
4. Level off higher

### Split-S
1. Entry: 250 KIAS (max)
2. Roll inverted
3. Pull through
4. Need 8,000+ ft
5. Exit much faster

### Aileron Roll
1. Entry: 350+ KIAS
2. Slight pull
3. Full stick deflection
4. Smooth 360°
5. Check altitude

### Cuban Eight
1. Entry: 450 KIAS
2. 5/8 loop up
3. Roll upright at 45° down
4. Pull to 45° up
5. Repeat other direction
