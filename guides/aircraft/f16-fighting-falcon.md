## Engine Start

### APU Start (Cold & Dark)

1. **Battery** - ON
2. **Main power** - BATT
3. **JFS (Jet Fuel Starter)** - START 2
4. **Wait** for JFS RPM
5. **Throttle** - IDLE (move from OFF)
6. **Monitor:**
   - RPM increasing
   - FTIT (turbine temp) rising
   - Oil pressure normal
7. **Engine stable** at idle (~65% RPM)
8. **JFS** - OFF (auto shutdown)
9. **Main power** - MAIN PWR
10. **Avionics** - Power up sequence

### Quick Start
- Ctrl+E for auto-start in most add-ons

## Takeoff

### Standard Takeoff

1. **Line up** - Runway centerline
2. **Brakes** - HOLD
3. **Throttle** - MIL (100%)
4. **Engine check** - Instruments green
5. **Brakes** - RELEASE
6. **Afterburner** - ENGAGE (optional)
7. **Nosewheel steering** - Maintain centerline
8. **Rotation** - 150-170 KIAS
9. **Rotate** - Smooth 10-15° pitch
10. **Gear** - UP at positive climb
11. **Accelerate** - 300 KIAS minimum

### Afterburner Takeoff
- 5 stages of afterburner
- Full AB for max performance
- Significantly shorter roll
- Use for short runways / heavy loads

### Takeoff Speeds (Approximate)

## Flight Characteristics

### The "Electric Jet"
- Fly-by-wire flight controls
- Computer limits G and AOA
- Very responsive handling
- Departure resistant

### Speed Reference

| Regime | Speed |
|--------|-------|
| Approach | 155-175 KIAS |
| Pattern | 250-300 KIAS |
| Cruise | 400-500 KIAS |
| Corner velocity | 440 KIAS |
| Max | Mach 2.0 |

### G-Awareness
- Onset rate: Rapid stick = rapid G
- CAT I limit: 9G at light weight
- Sustained: 6-7G typical
- G-suit inflation automatic

### Angle of Attack
- Optimal: 8-12° (maneuvering)
- AOA bracket on HUD
- Limiter prevents departure
- Override available (caution!)

## Air Combat Maneuvering

### Energy Management
The F-16 is an **energy fighter** - maintain speed!

**Corner Velocity:** 440 KIAS
- Maximum instantaneous turn rate
- Best for one-circle fights

**Sustained Turn:** 330 KIAS / 15,000 ft
- 9G sustained possible
- Best turn rate over time

### Basic Fighter Maneuvers

**High Yo-Yo:**
1. Opponent turning hard
2. Pull up, trade speed for altitude
3. Roll inverted at apex
4. Pull down behind opponent
5. Regain parameters

**Low Yo-Yo:**
1. Opponent above, crossing
2. Pull down, gain speed
3. Zoom up toward opponent
4. Arrive with energy advantage

**Scissors:**
1. Overshoot situation
2. Hard reversal
3. Force opponent to overshoot
4. Alternate reversals
5. Slowest wins

**Break Turn:**
1. Max G turn into threat
2. Drop flares/chaff
3. 7-9G, dump nose if needed
4. Keep visual

## Approach & Landing

### Overhead Pattern

1. **Initial** - 300 KIAS, 1,500 AGL
2. **Break** - 270° turn, speed brake
3. **Downwind** - Gear down <280 KIAS
4. **Base** - Turn to base
5. **Final** - 155-175 KIAS

### Final Approach

| Config | Speed |
|--------|-------|
| Light | 155 KIAS |
| Normal | 165 KIAS |
| Heavy | 175 KIAS |

### On-Speed AOA
- Use AOA bracket in HUD
- Bracket centered = on speed
- Don't chase airspeed, fly AOA

### Landing

1. **Threshold** - 50 ft, on speed
2. **Flare** - Minimal (fighter landing)
3. **Touchdown** - Main gear
4. **Aerodynamic braking** - Hold nose up
5. **Nosewheel** - Lower gently
6. **Brakes** - Apply as needed
7. **Speed brake** - Deploy

### Touch-and-Go
1. Touchdown
2. MIL power (or AB)
3. Check instruments
4. Rotate at Vr
5. Gear up

## Emergency Procedures

### Engine Fire
1. **Throttle** - OFF
2. **Fuel master** - OFF (if fire persists)
3. **Eject** - If uncontrollable

### Engine Failure
1. **Airspeed** - Best glide (200 KIAS)
2. **Zoom** - Trade speed for altitude
3. **Attempt restart:**
   - Throttle - OFF then IDLE
   - JFS - START
4. **No restart** - Eject above 2,000 ft AGL

### Hydraulic Failure
1. **Check HYD A & B**
2. **Manual reversion** - Stiffer controls
3. **Land immediately**
4. **Expect pitch/roll issues**

### Flight Control Malfunction
1. **Maintain control**
2. **Reduce speed**
3. **Check MPO** (manual pitch override)
4. **Land at nearest suitable field**

### Ejection

**Decision altitude:** 2,000 ft AGL uncontrolled

1. **Assume position** - Head back
2. **Handles** - Pull either ejection handle
3. **Sequence:** Canopy jettison → Seat fires

**Zero-Zero capable** - Works at 0 altitude, 0 speed

## Aerobatics

### Loop
1. Entry: 450 KIAS
2. Pull 4G
3. Over the top at ~150 KIAS
4. Pull through
5. Exit same altitude

### Aileron Roll
1. Entry: 350+ KIAS
2. Slight pitch up
3. Full lateral stick
4. Neutral at inverted
5. Continue to upright

### Cuban Eight
1. Entry: 400 KIAS
2. Half loop up
3. At 45° down, half roll
4. Pull to 45° up
5. Repeat other side

### Immelman
1. Entry: 450 KIAS
2. Pull vertical
3. At apex, roll 180°
4. Level flight, higher altitude

### Split-S
1. Entry: 250 KIAS (will gain speed)
2. Roll inverted
3. Pull through
4. Need 5,000+ ft
