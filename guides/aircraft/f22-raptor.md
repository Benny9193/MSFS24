## Engine Start Procedure

### Cold & Dark Start

1. **Battery** - ON
2. **APU** - START
3. **Wait for APU online**
4. **Canopy** - CLOSE
5. **Left Engine:**
   - Throttle - IDLE
   - Engine start switch - START
   - Monitor RPM, EGT
   - Stable at idle - CHECK
6. **Right Engine:**
   - Throttle - IDLE
   - Engine start switch - START
   - Monitor RPM, EGT
   - Stable at idle - CHECK
7. **APU** - OFF (after engines stable)
8. **Avionics** - ON
9. **Flight controls** - CHECK (full deflection)

### Quick Start (Ctrl+E)
- Auto-start available in most add-ons
- Engines spool to idle automatically

## Takeoff

### Standard Takeoff

1. **Line up** on runway centerline
2. **Brakes** - HOLD
3. **Throttle** - MIL (military power, 100%)
4. **Check engine instruments** - Both engines stable
5. **Brakes** - RELEASE
6. **Afterburner** - ENGAGE (if desired for short field)
7. **Rotation speed** - ~150-170 KIAS
8. **Rotate** - Smooth stick back
9. **Gear** - UP (positive rate of climb)
10. **Climb** - 350-400 KIAS initial

### Afterburner Takeoff
- Significantly shorter ground roll
- Higher fuel consumption
- Use for short runways or max performance

## Supercruise

The F-22's signature capability - sustained supersonic flight without afterburner.

### How to Achieve Supercruise

1. **Climb** to FL350-FL450
2. **Accelerate** using afterburner to M 1.5+
3. **Reduce throttle** to MIL power
4. **Maintain** M 1.5-1.8 on military power alone
5. **Altitude sweet spot**: 40,000-50,000 ft

### Benefits
- Dramatically extended range vs afterburner
- Lower heat signature
- Quieter operation

## Aerobatics & Maneuvering

### Basic Maneuvers

**Loop:**
1. Entry speed: 400+ KIAS
2. Pull 4-5 G
3. Maintain back pressure over top
4. Recovery: Same altitude, same heading

**Barrel Roll:**
1. Entry speed: 350+ KIAS
2. Pitch up 20-30°
3. Roll while maintaining pitch
4. Complete 360° roll

**Immelmann:**
1. Entry speed: 450+ KIAS
2. Pull into vertical
3. At apex, roll 180°
4. Level off (reversed heading, higher altitude)

**Split-S:**
1. Entry speed: 250 KIAS (will gain speed)
2. Roll inverted
3. Pull through
4. Need 8,000+ ft altitude to complete

### High-AOA Maneuvering

**Cobra (Pugachev's Cobra):**
1. Entry speed: 200-250 KIAS
2. Rapid full aft stick
3. Pitch to 90-120°
4. AOA reaches 90°+
5. Release stick, aircraft recovers
6. Dramatic speed reduction

**J-Turn:**
1. Entry speed: 300 KIAS
2. Pull into vertical
3. At low speed, full rudder
4. Pivot around vertical axis
5. Dive to regain speed

*Note: These maneuvers depend on flight model accuracy in your add-on*

## Climb & Descent

### Climb Profile

| Altitude | Speed | Power |
|----------|-------|-------|
| SL - 10,000 | 350-400 KIAS | MIL/AB |
| 10,000 - FL350 | 400-450 KIAS | MIL |
| FL350 - FL450 | M 0.85-0.90 | MIL |
| Above FL450 | M 0.90 | MIL |

**Zoom Climb:**
- Accelerate to M 1.5+ at lower altitude
- Pull into 45-60° climb
- Maintain afterburner
- Can reach 60,000+ ft

### Descent

**Normal Descent:**
- Idle power
- Speed brakes if needed
- 300-350 KIAS
- 4,000-6,000 ft/min descent rate

**Emergency Descent:**
- Idle power
- Speed brakes extended
- Maximum maneuvering speed
- Bank to increase descent rate

## Approach & Landing

### Traffic Pattern

1. **Initial** - 300 KIAS, 1,500 ft AGL
2. **Break** - 3-4 G turn at end of runway
3. **Downwind** - 200-250 KIAS, gear down
4. **Base** - 180 KIAS, configure
5. **Final** - 150-170 KIAS on speed

### Final Approach

| Item | Setting |
|------|---------|
| Speed | 150-170 KIAS |
| AOA | On-speed (bracket) |
| Gear | DOWN |
| Flaps | LAND |
| Throttle | As required |

### Landing

1. **Threshold crossing** - 50 ft, on speed
2. **Flare** - Minimal, fly onto runway
3. **Touchdown** - Main gear first
4. **Nosewheel** - Lower gently
5. **Aerodynamic braking** - Hold nose up
6. **Wheel brakes** - Apply progressively
7. **Speed brake** - Deploy if equipped

### Touch & Go

1. Land normally
2. Retract flaps to takeoff position
3. Apply full power / afterburner
4. Rotate at appropriate speed
5. Gear up, climb out

## Emergency Procedures

### Single Engine Failure

1. **Maintain control** - Wings level
2. **Identify** - Dead engine
3. **Throttle** - Good engine as needed
4. **Secure** - Failed engine if required
5. **Land** - At nearest suitable runway

### Dual Engine Failure

1. **Airspeed** - Best glide (~200 KIAS)
2. **Turn toward field**
3. **Attempt restart** - APU, windmill
4. **Eject** if unable to reach runway

### Landing Gear Malfunction

1. **Recycle** - Gear handle
2. **Emergency extension** - If equipped
3. **G-force** - Positive G may help
4. **Gear unsafe** - Land on foam if available

### Fire

1. **Throttle** - Affected engine to idle
2. **Fuel shutoff** - If fire persists
3. **Land immediately**
4. **Eject** - If fire uncontrollable

### Ejection (If Modeled)

1. **Assume position** - Head back, arms in
2. **Ejection handle** - PULL
3. Automatic sequence: Canopy jettison → Seat rockets

## Mission Profiles

### Combat Air Patrol (CAP)
```
Takeoff → Climb to FL350-400 → Supercruise to station
→ Loiter at M 0.85 → Return → Land
Duration: 2-3 hours with internal fuel
```

### Intercept
```
Takeoff → Afterburner climb → Supercruise to target
→ Engage → Return → Land
Emphasize: Speed and altitude advantage
```

### Airshow/Demo
```
Takeoff → Immediate afterburner climb
→ Series of high-G maneuvers
→ Low passes → High-AOA demos
→ Tactical landing
```
