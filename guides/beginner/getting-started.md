# Microsoft Flight Simulator 2024 - Getting Started Guide

## System Requirements

**Minimum:**
- OS: Windows 10/11 (64-bit)
- CPU: Intel i5-8400 / AMD Ryzen 5 1500X
- RAM: 16 GB
- GPU: NVIDIA GTX 970 / AMD RX 590 (4GB VRAM)
- Storage: 50 GB SSD (streaming) or 500GB+ (offline)
- Internet: 25 Mbps recommended

**Recommended:**
- CPU: Intel i7-12700K / AMD Ryzen 7 7800X3D
- RAM: 32 GB
- GPU: NVIDIA RTX 4070 / AMD RX 7800 XT (8GB+ VRAM)
- Storage: NVMe SSD

---

## Installation

1. **Via Microsoft Store / Xbox App**
   - Sign in with your Microsoft account
   - Purchase or access via Game Pass
   - Download and install (initial download ~50GB)

2. **Via Steam**
   - Purchase on Steam
   - Download through Steam client
   - Link Microsoft account when prompted

---

## First Launch Setup

1. **Graphics Settings** - Start with "Medium" preset and adjust
2. **Data Streaming** - Enable for best visuals (requires stable internet)
3. **Control Setup** - Configure your input device (keyboard, controller, or flight stick)
4. **Flight Training** - Complete the tutorials before free flight

---

---

## Performance Tips

1. **Lower LOD settings** if experiencing stutters
2. **Disable photogrammetry** in areas with poor internet
3. **Use DX12 mode** for modern GPUs
4. **Enable DLSS/FSR** if available for your GPU
5. **Limit frame rate** to reduce GPU load and heat
6. **Close background apps** to free RAM

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Long load times | Use SSD, check internet speed |
| CTD (Crash to Desktop) | Update GPU drivers, verify game files |
| Poor performance | Lower graphics settings, disable traffic |
| Black screens | Disable fullscreen optimizations |
| Controller not detected | Reconnect before launching game |

---

## Useful Keyboard Shortcuts

- `V` - Cycle camera views
- `F2` - Cockpit view
- `End` - External view
- `Ctrl+E` - Auto-start engines
- `B` - Toggle parking brake
- `L` - Toggle lights
- `Alt+Enter` - Toggle fullscreen
